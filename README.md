GENAID — Secure DNA Identity System
A conceptual forensic system design focused on securely collecting, storing, and matching DNA profiles at scale. The project emphasizes privacy-preserving architecture, ethical data handling, and scalable matching mechanisms to support modern criminal investigations.


Animal Zenith — AI-Based Animal Type Classification
An AI-driven system developed during Smart India Hackathon 2025 to automate Animal Type Classification of cattle and buffaloes using smartphone images. It applies computer vision techniques to measure body traits and generate standardized classification scores, designed for offline-first rural environments.


Smart Office Automation & Energy Management
A prototype automation system that reduces energy wastage by controlling lights and appliances based on motion detection. The project demonstrates computer vision–based occupancy detection, automatic device shutdown on inactivity, and basic energy usage logging for analysis.
